# fullSlider

Very basic horizontal slider on full width.

# Example

html:

```html
<div class="full-slider">
	<div class="inner">
		<div class="scroll-bar clearfix">
			<div class="item active">
				<img src="gfx/img.jpg" alt="" />
			</div>
			<div class="item">
				<img src="gfx/img.jpg" alt="" />
			</div>
			<div class="item">
				<img src="gfx/img.jpg" alt="" />
			</div>
			<div class="item">
				<img src="gfx/img.jpg" alt="" />
			</div>
			<div class="item">
				<img src="gfx/img.jpg" alt="" />
			</div>
		</div>
	</div>
</div>
```

js:

```javascript
$('.full-slider').fullSlider();
```
